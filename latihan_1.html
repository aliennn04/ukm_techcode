MY STORY

Hallo nama saya rizal ependi bisa di panngil rizal,ijong, alien. saya lahir pada tahun 2003 bulan september tanggal 08. alamat saya di montong sebie desa dakung,tapi kemarin pindah ke desa breaim. saya suka menonton anime,baca manga,manhua,manhwa genre kesukaan saya magic,isekai,kultivator,murim,system. keseharian saya kalau pagi masak nasi,cuci piring/baju kalau ada! setelah selesai melakukan yang tadi saya lanjut rebahan sambil jaga warung,dan baca manhwa saya bersekolah dasar di sdn dakung,mts darul ulum,smk alhasanain jurusan rekaya perangkat lunak dan sekarang alhamdulillah bisa lanjut kuliah di stmik lombok. awal masuk kuliah di tahun 2022 dan sekarang sudah semester 3,dan jurusan yang saya ambil adalah teknik informatika saya biasanya masuk kuliah jam 12.30,tapi karena saya mengikuti unit kegiatan mahasiswa technology code (ukm techcode) jadi saya ke kampus jam 07.30 dan pulang jam 17.--. kadang saya juga nginap di kampus untuk belajar,mabar,ngobrol,dan kumpul sama senior di UKM TECHCODE.
